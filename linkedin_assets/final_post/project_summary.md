# SquintScale: Project Summary

## Problem Statement
Reading digital or physical text can be a significant challenge for many. Visual strain from long sessions, accessibility barriers for users with dyslexia, and the difficulty of maintaining focus for those with ADHD often lead to reading fatigue and reduced comprehension. Traditional reading apps often lack the specialized focus and accessibility features needed to bridge this gap.

## Solution Overview
SquintScale is an accessibility-first reading assistant designed to empower every reader. By combining advanced neural simulation with practical productivity tools, SquintScale provides a tailored reading experience that adapts to the user's specific needs—whether they are scanning a physical page, uploading a document, or reading an article from a URL.

## Key Features
- **Camera Scanning**: Instant OCR for physical text documents.
- **Multi-Format Uploads**: Support for images and text documents.
- **URL Article Reader**: Clean, distraction-free reading for web content.
- **ADHD Focus Mode**: Guided reading markers to maintain focus.
- **Dyslexia Friendly Mode**: Specialized fonts and high-contrast themes.
- **Reading Analytics**: Insights into reading speed, focus time, and session history.
- **Dark Mode**: High-fidelity dark theme for reduced eye strain.

## Accessibility Impact
SquintScale moves beyond simple font changes. It implements WCAG 2.2 standards, offers vision simulation for testing color physiology, and provides a "glassmorphic" UI that prioritizes clarity and reduced cognitive load. It is designed to be a tool for inclusivity in the digital age.

## Tech Stack
- **Frontend**: React, Framer Motion, Tailwind CSS
- **Backend**: FastAPI (Python), PostgreSQL
- **Database/Auth**: Supabase
- **Infrastructure**: Docker, GitHub Actions
- **OCR/ML**: Neural-integrated text extraction
